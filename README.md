# Sorting

Sorting arrays in window

Used: Python2.7, Tkinter

python2.7 main.py [sorttype=quicksort|shaker_sort length]
  
  sorttype: quicksort (default)
            shaker_sort

  length:   100..1000 for quicksort (default 350)
            10 ..300  for shaker_sort


In program:
  q - exit
  w - shuffle array
  e - sort array
